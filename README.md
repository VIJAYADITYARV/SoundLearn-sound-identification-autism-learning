# Sound Identification Learning Platform for Autism

A React-based, autism-friendly learning platform designed to help children identify and understand everyday sounds.

## Features
- Sound identification using text-based cues
- Categories: Animals, Vehicles, Nature, Household, Human sounds
- Interactive modes: Exploration, Quiz, Matching, Memory
- Accessibility settings: Text size, high contrast, reduced motion
- Responsive and child-friendly UI

## Tech Stack
- React
- Tailwind CSS
- Local Storage

## Purpose
Built as part of a Full Stack Lab Evaluation with a focus on accessibility and inclusive learning.
